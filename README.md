# unmappedplaces.pl
_A fork of Gary68's unmappedplaces.pl script_

Finds unmapped places in OSM files

## Run

```bash
perl unmappedplaces.pl mexico-20150831.osm unmap.html unmap.slippy
```

You'll see the results in the *.html file generated.
